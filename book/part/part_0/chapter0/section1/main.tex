El objetivo fundamental de la minería de datos es \textbf{extraer conocimiento útil a partir de datos crudos y transformarlo en información accionable}. 

En términos prácticos, busca responder preguntas complejas para tomar decisiones estratégicas. Por ejemplo:
\begin{itemize}
    \item \textbf{Predecir el futuro:} ¿Qué clientes tienen más probabilidad de cancelar su suscripción el próximo mes?
    \item \textbf{Clasificar:} ¿Esta transacción bancaria es legítima o es un fraude?
    \item \textbf{Agrupar (Clustering):} ¿Cómo podemos dividir a nuestros compradores en segmentos con gustos similares para lanzar campañas personalizadas?
\end{itemize}
